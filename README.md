## 📱주요 기능

- **SnapKit**을 활용하여 코드로 오토레이아웃 구현
- **RxSwift**를 활용하여, 현재 날씨와 예보 요청 및 UI구현
- **Core Location**을 활용해서 실제 GPS 좌표로 날씨 데이터 요청
- **MVVM**패턴 적용
